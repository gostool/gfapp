### 1.2 log
日志路径:
* /tmp/log/gf-app/server
* /tmp/log/gf-app/debug

logger实例的名字:
* server
* 默认日志
* debug
```toml
[server]
	Address     = ":8299"
	ServerRoot  = "public"
	ServerAgent = "gf-app"
	LogPath     = "/tmp/log/gf-app/server"

# Logger.
[logger]
    Path        = "/tmp/log/gf-app"
    Level       = "all"
    Stdout      = true
    [logger.debug]
        path   = "/tmp/log/gf-app/debug"
        level  = "dev"
        stdout = true

```
日志的使用参考config/config_test.go 中的TestLogger
